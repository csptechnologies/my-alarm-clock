# my-alarm-clock
An alarm clock that makes you wake up no matter what. If you dont silence it within 45 seconds, it will start blaring horns and sirens!


Parts:

**Provided by Hack Club**
- Lolin C3 Mini ESP 32
- 2.25in TFT display
- 12 Keyboard Switches
- A 3.3V Piezo Buzzer
- Some Jumper Cables for wiring

**Parts needed**
- 5v 2-Channel Relay module

**Already Have**
- Wheelock Exceeder Horn Strobe
- System Sensor L Series LF Sounder
